Installing

    • Install Homebrew

      ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

      Copy/Paste SVG folder into directory

      Install Font Custom

      1. brew install fontforge eot-utils ttfautohint

      2. gem install fontcustom

      Run Font Custom

      fontcustom watch /path/to/vectors

      Include Font Custom CSS in project

      <link href="path/to/fontcustom/fontcustom.css" rel="stylesheet">
